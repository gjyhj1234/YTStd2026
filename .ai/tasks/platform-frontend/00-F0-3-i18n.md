## 核心功能清单

| 功能 | 说明 |
|------|------|
| import.meta.glob | 自动加载 views/components/layouts 下的 `*.vue.{locale}.json` |
| mergeComponentMessages | 过滤 null 值的合并函数 |
| 消息合并顺序 | common → generated → enums → 主语言文件 → 组件级（最高优先级） |
| vue-i18n 配置 | `legacy: false`、`globalInjection: true`、`locale: 'zh-CN'`、`fallbackLocale: 'en-US'` |
| 5 种语言支持 | zh-CN、en-US、ja-JP、ms-MY、zh-TW |
| null 值约定 | 组件级文件中 null 表示"由 common 提供" |
