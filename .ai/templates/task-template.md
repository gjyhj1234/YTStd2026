# 任务模板

使用本模板定义一个完整的任务描述，提供给 Agent 执行。

---

```markdown
# 任务：{任务标题}

## 任务信息

- **任务编号**：{编号，如 TASK-001}
- **所属阶段**：{阶段编号，如 Phase 3}
- **优先级**：{高/中/低}
- **预估时间**：{20-50 分钟}
- **前置任务**：{依赖的任务编号，无则填"无"}

## 任务目标

{一段话描述本任务要达成的目标}

## 前置阅读

执行本任务前，Agent 必须阅读以下文件：

- `.ai/system/agent-contract.md`
- `.ai/rules/global.md`
- {其他相关规则文件}
- {相关上下文文件}
- {相关业务文档}

## 输入

- {描述任务需要的输入，如"已完成的实体文件"、"数据字典"等}

## 预期输出

- {列出本任务的预期产出文件}
  - 文件 1：`{路径}` — {用途}
  - 文件 2：`{路径}` — {用途}

## 执行步骤

1. {具体步骤 1}
2. {具体步骤 2}
3. {具体步骤 3}
4. 执行编译验证：`dotnet build YTStd.slnx`
5. 执行测试验证：`dotnet test YTStd.slnx`

## 约束

- {约束 1}
- {约束 2}

## 禁止事项

- {禁止 1}
- {禁止 2}

## 验收标准

- [ ] {标准 1}
- [ ] {标准 2}
- [ ] 编译通过
- [ ] 已有测试通过
- [ ] 公开类型和方法有中文 XML 注释

## 会话结束时

完成本任务后，请按 `.ai/templates/session-summary-template.md` 输出会话总结。
```

---

## 使用说明

1. 复制本模板内容
2. 填写所有 `{占位符}` 内容
3. 将填写好的任务描述提供给 Agent
4. Agent 按执行步骤依次完成
5. Agent 完成后输出会话总结
