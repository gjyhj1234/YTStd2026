# 会话总结模板

每轮 Agent 会话结束时，必须按本模板输出会话总结。

---

```markdown
## 会话总结

### 会话信息
- **日期**：{YYYY-MM-DD}
- **轮次**：{第 N 轮}
- **任务编号**：{任务编号}
- **任务标题**：{任务标题}

### 当前所处阶段
- {阶段编号与名称}

### 本轮目标
- {本轮计划完成的内容}

### 本轮已完成

| 序号 | 操作 | 文件路径 | 说明 |
|-----|------|---------|------|
| 1 | 新建 | {路径} | {说明} |
| 2 | 修改 | {路径} | {说明} |

### 验证结果
- 编译：{✓ 通过 / ✗ 失败}
- 测试：{✓ N 个通过 / ✗ N 个失败}
- 测试命令：{实际执行的命令}

### 决策记录
- {在本轮中做出的设计决策及原因}

### 未完成内容
- {本轮未完成但计划在后续完成的内容}

### 风险与待确认
- {需要人类确认的问题}
- {发现的潜在风险}

### 下一轮应继续
- 从文件 `{文件路径}` 继续
- 下一步操作：{具体描述}

### 下一轮必须保持一致的规则
- {列出跨会话必须保持一致的命名、结构等}

### 下一轮建议阅读的文件
- {下一轮 Agent 应先阅读的文件}
```

---

## 使用说明

1. Agent 在每轮会话结束时使用本模板输出总结
2. 总结内容保存到 `.ai/workspace/session-summary-{YYYYMMDD}-{序号}.md`
3. 下一轮 Agent 开始时先阅读最新的会话总结
4. 人类可以在总结中添加审查反馈

## 保存位置

`.ai/workspace/session-summary-{YYYYMMDD}-{序号}.md`

示例：
- `.ai/workspace/session-summary-20260407-01.md`
- `.ai/workspace/session-summary-20260407-02.md`
