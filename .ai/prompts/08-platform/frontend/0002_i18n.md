## 执行步骤

1. 确认 `src/locales/index.ts` 已使用 `import.meta.glob` 自动加载组件级语言文件（参见 `.ai/prompts/05-i18n/frontend-i18n.md` 中的实现模板）
2. 执行 `dotnet build YTStd.slnx` 生成 `locales/generated/`
3. 确认 `locales/common/` 下有 5 种语言文件（zh-CN/en-US/ms-MY/zh-TW/ja-JP）
4. 为每个 `.vue` 组件创建紧贴的 5 个语言文件（`*.vue.zh-CN.json` 等）
5. 组件特有的翻译 key 写在组件级语言文件中（禁止放在主语言文件 `locales/{locale}.json`）
6. 复用文本在组件 JSON 中写 `null`（由 common 提供翻译）
7. 配置 DevExtreme 组件本地化
8. 实现语言切换（用户偏好 → 租户默认 → zh-CN）

- `views/PlatformOperation/` — 平台运营
