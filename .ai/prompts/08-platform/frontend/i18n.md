DevExtreme 组件本地化需要通过 dxdocs 查阅官方配置方案：

1. **调用 `devexpress_docs_search`**（使用 `technologies: ["Vue"]`）
2. **调用 `devexpress_docs_get_content`** 获取全文
3. **反思内容** 并基于检索到的信息编码

```

```

## 执行步骤

1. 实现 `src/locales/runtime/t.ts`、`gt.ts`、`loader.ts`
2. 执行 `dotnet build YTStd.slnx` 生成 `locales/generated/`
3. 创建 `locales/common/zh-CN.json` 和 `locales/common/en-US.json`
4. 为每个 `.vue` 组件创建紧贴的 `.vue.zh-CN.json` 和 `.vue.en-US.json`
5. 配置 DevExtreme 组件本地化
6. 实现语言切换（用户偏好 → 租户默认 → zh-CN）
